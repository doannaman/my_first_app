# My Learning Journey App
This application is a personal project created to apply and practice the concepts I have recently learned.
## What I Learned:
* **PyTorch:** Built a deep learning model from scratch and understood how core functions like `torch.nn.Flatten()` operate.
* **CustomTkinter & Tkinter:** Learned how to design and build a modern desktop graphical user interface (GUI).
* **PyInstaller:** Learned how to bundle and export a Python application into a standalone executable (`.exe`) file.
* **Github:** Learned to push the code and app to github, create readme and gitignore file
As a beginner, I look forward to building more advanced applications in the future to improve my skills.
> **NOTICE:** The model included in this app was built solely for learning purposes and may not perform with high accuracy. I apologize in advance for any incorrect classifications!
